# Sagewards

##  Local Setup

### Installation:
1. Java 21 has to be installed in the system.
2. Clone the repo

### to run in local

For Mac
```declarative
./mvnw clean install
./mvnw exec:java -Dexec.mainClass="com.sagewards.mcp.McpApplication"
```
For Windows
```declarative
mvnw.cmd clean install
mvnw.cmd exec:java -Dexec.mainClass="com.sagewards.mcp.McpApplication"
```
